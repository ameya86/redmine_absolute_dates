= Absolute Dates
This Plugin support Japanese environment only!

suerさんのプラグインをforkして、日時表示にしています。

= 機能
Redmineのチケットの作成日、更新日はデフォルトでは
「今日」からの相対的な日時数で表示されます。(...日前など)

このプラグインはこの日時を通常の日時(yyyy/MM/dd HH:MM)で表示します。

= インストール
1. Redmineのインストールディレクトリ内の vender/plugins/ にて

   $ git clone git://github.com/ameya86/redmine_absolute_dates.git

2. Redmine の再起動
